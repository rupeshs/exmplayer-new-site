Ubuntu Packages

To install exmplayer 5.0 on Ubuntu, just run these commands in a terminal:
Ubuntu 20.04 or above use the following commands
sudo add-apt-repository ppa:exmplayer-dev/exmplayer-qt5
sudo apt-get update
sudo apt-get install exmplayer
Older Ubuntu versions(<20.04) use the following commands
sudo add-apt-repository ppa:exmplayer-dev/exmplayer
sudo apt-get update
sudo apt-get install exmplayer
