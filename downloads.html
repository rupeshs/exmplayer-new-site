ExMplayer 2.0 packages for Windows and Linux

ExMplayer currently supports Windows XP/Windows Vista/Windows 7/Windows 8.
Download Installer
To install exmplayer on Ubuntu, just run these commands in a terminal:
sudo add-apt-repository ppa:exmplayer-dev/exmplayer 
sudo apt-get update 
sudo apt-get install exmplayer
You can get the latest source from Github with this command: git clone https://github.com/rupeshs/ExMplayer.git
